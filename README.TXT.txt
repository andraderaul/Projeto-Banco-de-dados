A aplicação está em um .jar dentro da pasta aplicacao/aplicacaoLemovie/Lemovie.jar

para executar a aplicação em um terminal linux siga os seguintes comandos 
$ java -jar nomeDaAplicacao
para rodar no terminal windows execute o seguite comando ou clique duas vezes
> java -jar nomeDaAplicacao

após rodar o .jar uma janela será mostrada na tela solicitando login e senha, por exemplo
Usuário: andrebritto
Senha: 12345

após logado aparecerá cinco botões de consultas basicas e mais um botão para 10 consultas especificas,
No caso as dez consultas solicitadas pela professor.

Ao clicar em um botão de consulta será exibido uma tabela com as informacoes referente a consulta, e 
em cada consulta existe um botão de voltar.

Mais informações especificas no Relatório.
